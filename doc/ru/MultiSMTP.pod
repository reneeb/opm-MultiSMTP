=head1 НАИМЕНОВАНИЕ

MultiSMTP - позволяет OTRS использовать несколько SMTP-серверов для отправки почты

=head1 ОПИСАНИЕ

Стандартная установка OTRS не поддерживает различные SMTP-серверы
для каждого адреса отправителя. Это может привести к проблемам, поскольку
сервер получателя отказывается доставлять почту от адресов которые не принадлежат отправляющему серверу.

Следовательно, каждому адресу отправителя нужен собственный SMTP-сервер, и
данный пакет предоставляет такую возможность.

После установки в блоке электронной почты в разделе администратора появится новый элемент.
Для управления адресом откройте менеджер SMTP настроек. Вы можете настроить
параметры для каждого адреса, указанного в системных адресах.
Также вы можете определить тип SMTP-соединения (простой SMTP, SMTPTLS или SMTP/S).
Пароль пользователя можно сохранить в зашифрованном виде - если вы хотите использовать эту функцию,
вам необходимо установить модули Perl C<Crypt::CBC> и C<Crypt::DES>.

Когда почта отправляется, пакет ищет параметры SMTP, принадлежащие адресу отправителя,
и если он ничего не может найти, он использует настройки SMTP из I<SysConfig>
(который используется стандартным механизмом SMTP).
